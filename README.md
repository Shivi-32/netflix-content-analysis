# Netflix Content Data Analysis

## Overview
This project performs exploratory data analysis on Netflix movies and TV shows to understand content distribution, trends, and genre patterns.

## Dataset
Netflix Movies and TV Shows dataset (public)

## Objectives
- Analyze Movies vs TV Shows distribution
- Study content growth over years
- Identify top contributing countries
- Explore common genres

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
